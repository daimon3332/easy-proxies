# Easy Proxies v1.3

## English

- Fixed subscription refresh progress staying at `0` while node testing was running.
- Added live phase, node totals, passed, failed, and promoted counters.
- Kept bounded asynchronous node testing and prevented overlapping subscription refresh jobs.
- Reworked refresh UI into a dismissible background task with a global status entry.
- Restored refresh polling after tab navigation or browser reload while the service is running.
- Added retry with backoff for temporary refresh status request failures.
- Prevented refresh completion from overwriting the page currently being viewed.
- Marked subscriptions whose entire node test failed as failed instead of successful.
- Added regression coverage, race testing, vet checks, and tagged Windows build verification.

## 简体中文

- 修复订阅刷新在节点测速期间长期显示 `0` 的问题。
- 增加实时刷新阶段、节点总数、已完成、成功、失败和入池统计。
- 保留有界异步节点测速，并阻止重复启动重叠的订阅刷新任务。
- 将刷新界面改为可收起的后台任务，并增加全局状态入口。
- 在服务仍运行时，切换 TAB 或刷新浏览器后自动恢复刷新进度。
- 临时网络错误增加带退避的状态轮询重试。
- 修复刷新完成后覆盖用户当前页面的问题。
- 当一个订阅的全部节点测速失败时，正确标记为失败而不是成功。
- 增加回归测试、竞态测试、vet 检查和指定 tags 的 Windows 构建验证。
