# v1.2 — SVG Favicon & Documentation Polish / SVG 网站图标与文档优化

> Easy Proxies v1.2 adds a native vector favicon based on the project logo and improves the WebUI preview documentation.
>
> Easy Proxies v1.2 新增基于项目 Logo 的原生矢量网站图标，并进一步优化 WebUI 预览文档。

---

## 🎨 SVG Favicon / SVG 网站图标

- Added a hand-built vector favicon based on the existing blue-green Easy Proxies logo.
- Preserved the circular proxy flow, six network nodes, central shield, and blue-green gradient.
- Embedded the SVG in the Go binary and served it directly from `/favicon.svg`.
- Added the SVG favicon declaration to the WebUI document head.
- 新增基于现有蓝绿色 Easy Proxies Logo 绘制的矢量网站图标。
- 保留环形代理流、六个网络节点、中央盾牌和蓝绿渐变等视觉元素。
- 将 SVG 嵌入 Go 可执行文件，并通过 `/favicon.svg` 直接提供。
- 在 WebUI 文档头部增加 SVG Favicon 声明。

---

## 🧪 Validation / 验证

- Added an HTTP test for the favicon response status, MIME type, cache policy, and SVG content.
- Kept the favicon as a standalone vector asset without embedding raster image data.
- 增加 HTTP 测试，验证 Favicon 的状态码、MIME 类型、缓存策略和 SVG 内容。
- Favicon 使用独立矢量资源，不包含嵌入式位图数据。

---

## 🖼️ Documentation / 文档

- Kept the WebUI preview as a level-two README section.
- Restored default-collapsed screenshots while preserving the single-column layout.
- Synchronized the layout across English, Simplified Chinese, and Traditional Chinese READMEs.
- WebUI 预览继续使用 README 二级标题。
- 截图恢复为默认折叠，同时保留单列展示方式。
- 同步更新英文、简体中文和繁体中文 README 布局。
