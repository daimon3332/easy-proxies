# Easy Proxies v1.6

## English

- Fixed Clash YAML Shadowsocks nodes losing `plugin`, `plugin-opts`, and `udp-over-tcp` during import.
- Mapped Mihomo `plugin: obfs` configurations to sing-box's built-in `obfs-local` implementation.
- Preserved TLS simple-obfs mode and host values exactly, including host values containing a port suffix.
- Added sing-box UDP-over-TCP options for compatible Shadowsocks nodes.
- Removed the default `local` Tag from the import form.
- Added a prominent dialog and backend validation when an import Tag is empty.
- Added parser, outbound-builder, Tag-validation, race, and tagged-build regression checks.

## 简体中文

- 修复 Clash YAML Shadowsocks 节点在导入时丢失 `plugin`、`plugin-opts` 和 `udp-over-tcp` 的问题。
- 将 Mihomo 的 `plugin: obfs` 配置映射到 sing-box 内置的 `obfs-local` 实现。
- 完整保留 TLS simple-obfs 的模式和 Host，包括带端口后缀的 Host 值。
- 为兼容的 Shadowsocks 节点补充 sing-box UDP over TCP 配置。
- 移除导入界面默认填写的 `local` Tag。
- Tag 为空时增加明显弹窗，并由后端同步校验。
- 增加解析器、出站构建、Tag 校验、竞态测试和完整标签构建检查。
