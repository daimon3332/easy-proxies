# v1.0 — Speed Test, Stability & UI Overhaul / 测速、稳定性与 UI 全面升级

> The first public Easy Proxies release focuses on accurate node testing, a subscription-first workflow, responsive batch operations, and a redesigned WebUI.
>
> Easy Proxies 首个公开版本，重点改进节点测速准确性、订阅优先的操作流程、批量任务响应速度和 WebUI 体验。

---

## ✨ Highlights / 版本亮点

- Import subscription URLs, proxy URI lists, Base64 content, and Clash/Mihomo YAML.
- Test imported nodes concurrently with adaptive worker counts and visible progress.
- Automatically add passed nodes to the runtime pool after an import.
- Use `multi-port` mode by default, assigning an independent local port to every available node.
- 支持导入订阅 URL、代理 URI 列表、Base64 内容和 Clash/Mihomo YAML。
- 使用自适应并发 Worker 测试节点，并显示实时任务进度。
- 导入后可自动把测速通过的节点加入运行节点池。
- 默认使用 `multi-port` 模式，为每个可用节点分配独立本地端口。

---

## 🚀 Speed Testing / 节点测速

- Test real proxy connectivity through sing-box instead of relying on a basic TCP connection.
- Use `generate_204` endpoints for lightweight and consistent health probes.
- Select either `https://www.gstatic.com/generate_204` or `https://cp.cloudflare.com/generate_204` from Settings.
- Run batch tests as cancellable background jobs so long node lists do not block page navigation.
- 通过 sing-box 测试真实代理连通性，而不是只进行基础 TCP 连接判断。
- 使用 `generate_204` 地址进行轻量且一致的健康探测。
- 可在设置中选择 `https://www.gstatic.com/generate_204` 或 `https://cp.cloudflare.com/generate_204`。
- 批量测速以可取消的后台任务运行，处理大量节点时不阻塞页面切换。

---

## 🎨 WebUI and Workflow / 界面与操作流程

- Redesigned navigation, cards, buttons, form controls, status feedback, and responsive layouts.
- Reorganized Settings into clear runtime, port, pool, subscription, logging, and GeoIP sections.
- Enlarged interactive controls and selection targets for easier operation.
- Blacklist duration and rotation interval now use numeric seconds in the WebUI.
- 重做导航、卡片、按钮、表单控件、状态反馈和响应式布局。
- 设置页按运行、端口、节点池、订阅、日志和 GeoIP 分类展示。
- 放大按钮和选择控件的可点击区域，降低操作难度。
- 黑名单时间和轮换间隔在 WebUI 中统一使用数字秒数。

---

## 🔌 Proxy and Pool Management / 代理与节点池管理

- Support common protocols including VLESS, VMess, Trojan, Shadowsocks, ShadowsocksR, Hysteria, Hysteria2, TUIC, AnyTLS, HTTP/HTTPS, SOCKS4, and SOCKS5.
- Skip occupied local ports automatically and expose the actual assignments in the WebUI.
- Provide `multi-port`, `pool`, and `hybrid` runtime modes.
- Manage passed, failed, pooled, excluded, and imported nodes from one interface.
- 支持 VLESS、VMess、Trojan、Shadowsocks、ShadowsocksR、Hysteria、Hysteria2、TUIC、AnyTLS、HTTP/HTTPS、SOCKS4 和 SOCKS5 等常用协议。
- 自动跳过已占用的本地端口，并在 WebUI 中显示实际端口分配。
- 提供 `multi-port`、`pool` 和 `hybrid` 三种运行模式。
- 在同一界面管理测速通过、失败、已入池、已排除和已导入节点。

### Quick start / 快速开始

Windows:

```powershell
Copy-Item config.example.yaml config.yaml
.\easy_proxies.exe -config config.yaml
```

Linux:

```bash
cp config.example.yaml config.yaml
chmod +x easy_proxies
./easy_proxies -config config.yaml
```

Open `http://127.0.0.1:9091`, import subscription URLs, test nodes, and use the generated local ports.

打开 `http://127.0.0.1:9091`，导入订阅 URL、测试节点，然后使用系统生成的本地端口。
